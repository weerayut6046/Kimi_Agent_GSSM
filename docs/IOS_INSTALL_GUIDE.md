## 🎯 แนะนำระบบ

แอพ **Gas Station Shift Manager** รองรับการติดตั้งบน iPhone และ iPad ได้ **2 วิธี**:

| วิธี | ความยาก | ค่าใช้จ่าย | ข้อดีหลัก |
|------|---------|------------|-----------|
| **PWA** (Add to Home Screen) | ⭐ ง่ายมาก | ฟรี | ติดตั้งเองได้ทันที ไม่ต้องรอ App Store |
| **Native App** (App Store) | ⭐⭐⭐ ยาก | $99/ปี | อยู่บน App Store จริง Push Notification ได้เต็มที่ |

> 💡 **แนะนำ**: ถ้าคุณเป็นเจ้าของปั้มหรือผู้จัดการ ให้ใช้ **วิธีที่ 1 (PWA)** ก่อน เพราะติดตั้งได้ทันที ใช้งานได้เหมือนแอพจริง ไม่ต้องเสียค่าธรรมเนียม Apple

## 📱 วิธีติดตั้งแบบที่ 1: PWA (แนะนำ)

PWA (Progressive Web App) คือเว็บแอพที่ติดตั้งบน Home Screen ได้เหมือนแอพจริง ใช้งานได้แบบเต็มหน้าจอ ไม่ต้องผ่าน App Store

### ขั้นตอนการติดตั้ง

#### บน iPhone (iOS 16 ขึ้นไป)

1️⃣ **เปิด Safari** แล้วเข้าไปที่เว็บไซต์ของระบบ  
(เช่น `https://your-domain.com` หรือ URL ที่แอดมินให้มา)

2️⃣ **แตะปุ่มแชร์** (Share) ที่แถบด้านล่าง  
🔲 รูปสี่เหลี่ยมพร้อมลูกศรชี้ขึ้น

3️⃣ **เลื่อนลง** แล้วแตะ **"Add to Home Screen"**  
(เพิ่มลงหน้าจอหลัก)

4️⃣ **แตะ "Add"** (เพิ่ม) ที่มุมขวาบน

5️⃣ ✅ **เสร็จแล้ว!** ไอคอนแอพจะปรากฏบน Home Screen

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│  🌐 เปิดเว็บ    │  →  │  📤 แตะ Share   │  →  │  ➕ Add to Home │
│                 │     │                 │     │                 │
│  ┌───────────┐  │     │  ┌───────────┐  │     │  ┌───────────┐  │
│  │           │  │     │  │           │  │     │  │  [แอพ]    │  │
│  │  หน้าแอพ  │  │     │  │  รายการ   │  │     │  │  Gas Sta- │  │
│  │           │  │     │  │  แชร์...  │  │     │  │  tion     │  │
│  └───────────┘  │     │  └───────────┘  │     │  └───────────┘  │
│                 │     │     Add to Home │     │      [Add]      │
└─────────────────┘     └─────────────────┘     └─────────────────┘
```

#### บน iPad (iPadOS)

1. เปิด Safari → เข้าเว็บไซต์
2. แตะปุ่ม **Share** ที่มุมขวาบน (รูปสี่เหลี่ยม+ลูกศร)
3. เลือก **"Add to Home Screen"**
4. แตะ **"Add"**

### ลักษณะที่แสดงว่าติดตั้งสำเร็จ

- มีไอคอนแอพบน Home Screen
- เปิดแอพแล้วไม่มีแถบที่อยู่ (Address Bar) ของ Safari
- แสดงผลแบบเต็มหน้าจอเหมือนแอพจริง
- มี Splash Screen (หน้าจอเปิดตัว) สีน้ำเงินพร้อมโลโก้

### การลบแอพ PWA

1. กดค้างไอคอนแอพบน Home Screen
2. แตะ **"Remove App"** (ลบแอพ)
3. เลือก **"Remove from Home Screen"** (ลบออกจากหน้าจอหลัก)

> ⚠️ การลบ PWA จะไม่ลบข้อมูลในระบบ เพราะข้อมูลอยู่บน Cloud (Supabase)

## 🍎 วิธีติดตั้งแบบที่ 2: Native iOS App

สำหรับองค์กรที่ต้องการแอพบน App Store จริง ๆ

### สิ่งที่ต้องเตรียม

| รายการ | รายละเอียด |
|--------|-----------|
| 💻 **Mac Computer** | จำเป็นต้องใช้ MacBook หรือ iMac |
| 🛠️ **Xcode** | ติดตั้งจาก Mac App Store (ฟรี) |
| 🆔 **Apple Developer Account** | $99/ปี (~3,600 บาท) |
| 📧 **Apple ID** | สำหรับลงชื่อเข้าใช้ Xcode |

### ขั้นตอนสำหรับนักพัฒนา (Developer)

#### ขั้นตอนที่ 1: Build แอพ

```bash
# 1. เข้าไปที่โฟลเดอร์โปรเจค
cd gas-station-shift-manager

# 2. Build แอพ (สร้างไฟล์ production ใน dist/)
npm run build

# 3. ส่งไฟล์เข้า iOS Project
npm run sync:ios
# หรือ: npx cap sync ios

# 4. เปิด Xcode
npm run open:ios
# หรือ: npx cap open ios
```

#### ขั้นตอนที่ 2: ตั้งค่าใน Xcode

1. ใน Xcode ที่เปิดขึ้นมา เลือกโปรเจค **"App"** จาก sidebar ด้านซ้าย
2. ที่แท็บ **"Signing & Capabilities"**:
   - เลือก **Team** เป็น Apple Developer Team ของคุณ
   - เปลี่ยน **Bundle Identifier** ถ้าต้องการ (เช่น `com.yourcompany.gasstation`)
3. ที่แท็บ **"General"**:
   - ตั้งค่า **Version** (เช่น 1.0.0)
   - ตั้งค่า **Build** (เช่น 1)

#### ขั้นตอนที่ 3: Build และทดสอบ

1. เชื่อมต่อ iPhone เข้ากับ Mac
2. ที่ Xcode เลือกอุปกรณ์ (Device) เป็น iPhone ที่เชื่อมต่อ
3. กดปุ่ม **Run** (▶️) หรือ `Cmd + R`
4. ทดสอบใช้งานบน iPhone จริง

#### ขั้นตอนที่ 4: ส่งขึ้น App Store

1. ใน Xcode เลือกเมนู **Product → Archive**
2. รอจนกว่าจะ Build เสร็จ ( Organizer จะเปิดขึ้นมา )
3. แตะปุ่ม **"Distribute App"**
4. เลือก **"App Store Connect"**
5. เลือก **"Upload"**
6. รอตรวจสอบและ Upload (อาจใช้เวลา 10-30 นาที)

#### ขั้นตอนที่ 5: รอ Review จาก Apple

1. เข้าไปที่ [App Store Connect](https://appstoreconnect.apple.com)
2. ไปที่แอพของคุณ → เลือก Build ที่อัปโหลด
3. กรอกข้อมูลแอพ (คำอธิบาย, ภาพหน้าจอ, ราคา)
4. ส่ง **"Submit for Review"**
5. รอ Apple ตรวจสอบ (ปกติ 1-3 วันทำการ)

```
┌──────────────┐    ┌──────────────┐    ┌──────────────┐    ┌──────────────┐
│  npm run     │ →  │   Xcode      │ →  │   Archive    │ →  │   App Store  │
│  build:ios   │    │   Build      │    │   + Upload   │    │   Review     │
└──────────────┘    └──────────────┘    └──────────────┘    └──────────────┘
     10 นาที            5 นาที             15 นาที            1-3 วัน
```

## 🎮 การใช้งานเบื้องต้นบน iOS

### การเข้าสู่ระบบครั้งแรก

1. เปิดแอพจาก Home Screen
2. กรอก **Email** และ **Password**
3. แตะปุ่ม **"เข้าสู่ระบบ"**

> 🔑 ข้อมูล Login Demo:
> - Admin: `admin@gasstation.com` / `Admin@123`
> - Manager: `manager@gasstation.com` / `Manager@123`
> - Staff: `somchai@gasstation.com` / `Staff@123`

### การใช้งาน Offline (ไม่มีอินเทอร์เน็ต)

แอพรองรับการใช้งานแบบ Offline:

- ✅ ดูข้อมูลที่เคยโหลดไว้
- ✅ บันทึกการขาย POS (จะส่งขึ้น Cloud อัตโนมัติเมื่อมีเน็ต)
- ✅ เช็คอิน/เช็คเอาท์ กะงาน
- ✅ บันทึกบัญชีประจำวัน

> 📶 เมื่อมีสัญญาณอินเทอร์เน็ตกลับมา ข้อมูลจะซิงค์อัตโนมัติ

### การใช้งานบน iPad

- แอพรองรับหน้าจอ iPad แบบเต็มรูปแบบ
- แนะนำให้ใช้แนวนอน (Landscape) สำหรับดูรายงานและตารางข้อมูล
- รองรับ Split View (ใช้งานพร้อมแอพอื่นได้)

## 🔄 การอัปเดตแอพ

### กรณีใช้ PWA (Add to Home Screen)

**อัปเดตอัตโนมัติ!** เมื่อแอดมินอัปเดตระบบ:

1. เปิดแอพ
2. ถ้ามีเวอร์ชันใหม่ แอพจะโหลดอัตโนมัติในพื้นหลัง
3. ปิดแอพแล้วเปิดใหม่ จะได้เวอร์ชันใหม่ทันที

> 💡 **บังคับอัปเดต**: ถ้าอยากให้แน่ใจว่าเป็นเวอร์ชันล่าสุด ให้ลบแอพออกจาก Home Screen แล้วติดตั้งใหม่

### กรณีใช้ Native App (App Store)

1. เปิด **App Store**
2. ไปที่แท็บ **"Updates"** หรือค้นหาแอพชื่อ "Gas Station Shift Manager"
3. แตะ **"Update"**
4. รอดาวน์โหลดเสร็จ

## 🛠️ การแก้ไขปัญหาเบื้องต้น

### ❌ ปัญหา: เปิดแอพแล้วขึ้นหน้าขาว

| สาเหตุ | วิธีแก้ไข |
|--------|----------|
| ไม่มีอินเทอร์เน็ต และไม่เคยเปิดแอพมาก่อน | เชื่อมต่อ WiFi หรือ 4G/5G แล้วเปิดใหม่ |
| Service Worker ค้าง | ลบแอพออกจาก Home Screen → ติดตั้งใหม่ |

### ❌ ปัญหา: Login ไม่ได้

| สาเหตุ | วิธีแก้ไข |
|--------|----------|
| ลืมรหัสผ่าน | ติดต่อแอดมินให้รีเซ็ตรหัสผ่าน |
| ขึ้นข้อความ "Account locked" | รอ 15 นาที แล้วลองใหม่ |
| ขึ้นข้อความ "Network Error" | ตรวจสอบอินเทอร์เน็ต หรือรอสักครู่ |

### ❌ ปัญหา: ข้อมูลไม่อัปเดต

1. ปิดแอพแล้วเปิดใหม่ (Swipe up → ปัดแอพขึ้น → เปิดใหม่)
2. ถ้ายังไม่ได้ ให้ **Clear Safari Cache**:
   - ไปที่ **Settings → Safari → Clear History and Website Data**
   - หมายเหตุ: จะลบข้อมูลทั้งหมดของ Safari

### ❌ ปัญหา: แจ้งเตือน (Notification) ไม่มา

- **PWA บน iOS**: Apple ไม่อนุญาต Push Notification สำหรับ PWA ในขณะนี้
- **Native App**: ตรวจสอบว่าเปิด Notification ใน **Settings → Notifications → Gas Station Shift Manager**

### ❌ ปัญหา: แอพค้าง/ช้า

1. ปิดแอพแล้วเปิดใหม่
2. รีสตาร์ท iPhone (กดปิดเครื่องแล้วเปิดใหม่)
3. ถ้ายังช้า ให้ลบแล้วติดตั้งใหม่

### ❌ ปัญหา: หน้าเว็บค้างที่ Loading นาน

**สาเหตุหลัก:** Browser extensions บล็อกการเชื่อมต่อ Supabase (เช่น Bitwarden, AdBlock, Privacy Badger)

**แก้ไข:**
1. ลองเปิดใน **Safari Private mode** ถ้าเร็วขึ้น → แสดงว่า extensions เป็นตัวบล็อก
2. ไปที่ **Settings → Safari → Extensions** → ปิด extensions ที่น่าสงสัย
3. ลองปิด WiFi แล้วเปิดใหม่
4. ลบแอพออกจาก Home Screen แล้วติดตั้งใหม่

**สาเหตุรอง:** ไม่มี database index

**แก้ไข:** ติดต่อแอดมินให้สร้าง index ใน Supabase:
```sql
CREATE INDEX IF NOT EXISTS idx_users_authuid ON users(authuid);
CREATE INDEX IF NOT EXISTS idx_users_email ON users(email);
```
